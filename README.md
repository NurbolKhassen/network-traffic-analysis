# Network Traffic Analysis Lab (SOC Project)

## 📌 Overview

This project is a basic SOC (Security Operations Center) and network analysis lab created using Wireshark.  
The goal is to capture, analyze, and understand real network traffic generated by different protocols such as DNS, ICMP, HTTP, HTTPS (TLS), and TCP handshake.

This lab simulates real-world traffic monitoring tasks performed by SOC analysts.

---

## 🎯 Objectives

- Capture live network traffic using Wireshark
- Generate different types of network requests
- Analyze packets for multiple protocols
- Understand how normal network communication looks
- Identify key network behaviors used in cybersecurity monitoring

---

## 🛠 Tools Used

- Wireshark
- Windows CMD (nslookup, ping, curl)
- Web browser
- GitHub for documentation

---

## 🌐 Captured Network Traffic

### 1. DNS Traffic
- Tool used: `nslookup google.com`, `nslookup github.com`
- Purpose: Domain name resolution
- Observed in Wireshark using filter: `dns`

### 3. HTTP Traffic

### 4. HTTPS / TLS Traffic
